# Transaction-Fraud-Detection_And-Risk-Intelligence

## Overview

A SQL-based fraud detection and risk intelligence project that analyzes transaction data to identify anomalies, detect suspicious behavior, and build risk profiles for customers, merchants, and channels.

## Objectives
- Detect suspicious transaction behavior
- Build fraud risk scoring logic
- Generate actionable insights for risk monitoring

## Tools Used
- PostgrSQL 
- Power BI

## Key Features
- Fraud pattern detection queries
- Risk profiling (customer, merchant, channel)
- Executive-level reporting and insights
